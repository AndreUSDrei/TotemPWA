Arquitetura do Sistema
- O totem do ADS Burguer é um sistema web de autoatendimento, focado na interface para o usuário. Ele usa o padrão MVC para organizar melhor o código e facilitar o desenvolvimento. O controle de versões é feito com Git, seguindo o fluxo Gitflow para ajudar no trabalho em equipe.

Tecnologias Utilizadas no Totem ADS Burguer:
- Frontend:** HTML (CSHTML), CSS, JavaScript
- Linguagem:** C#
- Controle de Versão: Git (Gitflow)
- Framework Web: ASP.NET Core MVC
